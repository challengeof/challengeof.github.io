JRebel Mac破解

重要提示–最佳人生

  • 只推荐当计算机无法访问互联网时使用本破解文件。

  • 如果可以访问互联网，建议直接到JRebel官网注册JRebel会员获取【正版永久免费】使用的授权码。JRebel会员注册以关联FaceBook或Twitter账号登录的形式进行注册（类似QQ关联登录），关联登录并完善基本信息后即可获取正式授权码，基本获取流程如下：

、注册并登录FaceBook或Twitter账号（注册和登录都需要科学上网工具，关联登录JRebel后则不再需要使用科学上网工具）。

2、打开https://my.jrebel.com，选择对应的关联登录方式进行JRebel会员登录。
  • 、确认授权JRebel读取Twitter的相关信息
  • 、完善相关的信息
  • 、查看并复制授权码
  • 、启动IDEA，在Configure-Settings-JRebel-Open activation dialog中将复制的授权码粘贴到Activation Code

　　以上是”最佳人生”提供的正版获取方法.笔者同样推荐使用正版.

、Mac破解方法:

-　　注意:笔者教程中使用的版本是6.2.3

-　　1:在Idea中安装JRebel插件,并重启Idea.

-　　2:下载JRebel破解jar包.链接: http://pan.baidu.com/s/1mgkunWG 密码: 6jhb

-　　3:解压JRebel得到jrebel.jar和jrebel.lic.

-　　4:将解压得到的jrebel.jar和jrebel.lic复制到/Users/home/Library/Application Support/IntelliJIdea14/jr-ide-idea/lib/jrebel和/Users/home/Library/Application Support/IntelliJIdea14/jr-ide-idea/lib/jrebel6目录下.注意:复制之前最好备份之前的jrebel.jar.

-　　5:再次启动Idea并且到Jrebel–>Advanced中选择JRebel 6 Agent.点击Apply–>OK.

  • 现在你就可以使用JRebel热部署喽.

什么是 Cmd Markdown

您可以使用很多工具书写 Markdown，但是 Cmd Markdown 是这个星球上我们已知的、最好的 Markdown 工具——没有之一 ：）因为深信文字的力量，所以我们和你一样，对流畅书写，分享思想和知识，以及阅读体验有极致的追求，我们把对于这些诉求的回应整合在 Cmd Markdown，并且一次，两次，三次，乃至无数次地提升这个工具的体验，最终将它演化成一个 编辑/发布/阅读 Markdown 的在线平台——您可以在任何地方，任何系统/设备上管理这里的文字。

1. 实时同步预览

我们将 Cmd Markdown 的主界面一分为二，左边为编辑区，右边为预览区，在编辑区的操作会实时地渲染到预览区方便查看最终的版面效果，并且如果你在其中一个区拖动滚动条，我们有一个巧妙的算法把另一个区的滚动条同步到等价的位置，超酷！

2. 编辑工具栏

也许您还是一个 Markdown 语法的新手，在您完全熟悉它之前，我们在 编辑区 的顶部放置了一个如下图所示的工具栏，您可以使用鼠标在工具栏上调整格式，不过我们仍旧鼓励你使用键盘标记格式，提高书写的流畅度。

3. 编辑模式

完全心无旁骛的方式编辑文字：点击 编辑工具栏 最右测的拉伸按钮或者按下 Ctrl + M，将 Cmd Markdown 切换到独立的编辑模式，这是一个极度简洁的写作环境，所有可能会引起分心的元素都已经被挪除，超清爽！

4. 实时的云端文稿

为了保障数据安全，Cmd Markdown 会将您每一次击键的内容保存至云端，同时在 编辑工具栏 的最右侧提示 已保存 的字样。无需担心浏览器崩溃，机器掉电或者地震，海啸——在编辑的过程中随时关闭浏览器或者机器，下一次回到 Cmd Markdown 的时候继续写作。

5. 离线模式

在网络环境不稳定的情况下记录文字一样很安全！在您写作的时候，如果电脑突然失去网络连接，Cmd Markdown 会智能切换至离线模式，将您后续键入的文字保存在本地，直到网络恢复再将他们传送至云端，即使在网络恢复前关闭浏览器或者电脑，一样没有问题，等到下次开启 Cmd Markdown 的时候，她会提醒您将离线保存的文字传送至云端。简而言之，我们尽最大的努力保障您文字的安全。

我的童年的爱情

我的童年的爱情

风调雨顺了上千年的古都依然风调雨顺，浑浑噩噩了千百年的人们依然浑浑噩噩，在那个封建与开放并存，计划生育猖獗的一九xx年，我出生在张北！

虽然我出生在张北，但对这座城市的印象主要来自我的高中时期，我不知道我是出生多久以后才被妈妈带回了家乡——蔚县。
因为当时计划生育猖獗，所以在国家的政策上，我是不允许被生出来的，但是当时人们的思想比较封建，我的父母也不例外，所以才有了我！
长大后，常听妈妈说小时候的我是如如何如听话多么多么乖巧，其实我知道妈妈是在责怪现在我的叛逆，我并不反驳！
我想，我们那个时代的孩子，我是属于比较早熟的，因为我在十一岁的时候就‘’恋爱‘’了，直到现在，我都搞不清楚当时心里的那种感觉，是纯粹的爱慕？喜欢？还是别的什么的？～～当然，这已经不重要了，感谢她在十一岁那年，让我有着一些记忆！
到五年级的时候，我遇到了另外一个女孩儿，在如今的小说或肥皂剧里才有的一些情节，竟然发生在那个时候的我的身上，
记得是一个中午，我被班上的一个男孩带着几个比我们大的孩子拦在了回家的路上，而且来者不善，当时我的胆子和我的个子一样小，
明知道打不过也不敢跑，后来他说的话竟然把我逗乐了，他说让我和这个女孩儿分手，我恍然大悟，原来是情敌呀，当时我很愤怒，干嘛？抢婚呀，这玩意儿也能强迫？
哼，我当时就说了俩字儿，不行，现在想起来还觉得当时的自己还是挺勇敢的，在爱情面前，不畏权势，不付权威，不贪权利.
呵呵～～有时候，躺在被窝里，看着天花板，回忆一下童年，那个时候的我们是纯粹的，不管是对友情，还是爱情！
